# Iteration 11 — Система уровней + динамический спаун препятствий

## Что изменилось с предыдущей итерации
- Добавлен `LevelConfig.cs` — данные уровня (голы для победы, список препятствий)
- Добавлен `LevelDatabase.cs` — 5 уникальных конфигов + 25 уровней копирующих 1-5 рандомно
- Добавлен `ObstacleSpawner.cs` — динамический спаун препятствий по режиму
- Добавлен `LevelUI.cs` — UI прогресса уровня (голы забитые / нужные)
- Добавлен `LevelEndUI.cs` — экран победы/проигрыша уровня (RETRY/NEXT/MENU)
- Обновлён `MatchManager.cs` — GameMode.Levels: забил X голов = победа, пропустил 1 = проиграл, прогресс в PlayerPrefs
- Обновлён `GameStarter.cs` — поддержка уровней, спаун препятствий, StartLevel()
- Статичные препятствия из итерации 6 теперь спавнятся динамически (для Match/Endless — те же, для Levels — уникальные)

## Как настроить
1. Скопируй файлы в проект (заменить MatchManager.cs, GameStarter.cs)
2. Открой Game сцену
3. В Unity: меню **SoccerGame → Setup Level System - Game (Iteration 11)**
4. Сохрани Game сцену

## Что создаётся/меняется на сцене
- Дети объекта **Obstacles** удалены (будут спавниться динамически)
- **ObstacleSpawner** — объект со ссылками на bounce material и sprite
- **LevelPanel** на GameCanvas — прогресс уровня (скрыт по умолчанию)
- **LevelEndHandler** → **LevelEndPanel** — экран результата уровня (на всегда-активном родителе)

## Как тестировать
### Match/Endless (без изменений в поведении)
1. Запусти через меню — препятствия спавнятся такие же как раньше

### Levels (пока только через Inspector)
1. На MatchManager поставь Game Mode = Levels
2. В GameStarter (если есть) поставь selectedLevel = 1-5
3. Каждый уровень имеет свои препятствия и кол-во голов
4. Пропустил 1 гол = проиграл (LEVEL FAILED + RETRY/MENU)
5. Забил нужное кол-во = победил (LEVEL COMPLETE + NEXT/RETRY/MENU)

В итерации 12 будет UI выбора уровней и кнопка LEVELS в главном меню.

## 5 уникальных уровней
1. Level 1 — 2 гола, 3 статичных блока
2. Level 2 — 2 гола, 1 двигающийся + 3 статичных
3. Level 3 — 3 гола, 1 вращающийся + 1 двигающийся + 2 статичных
4. Level 4 — 3 гола, 2 двигающихся + 1 вращающийся + 3 статичных
5. Level 5 — 4 гола, 2 вращающихся + 2 двигающихся + 1 статичный

## Ожидаемый результат
Система уровней с уникальными препятствиями, динамический спаун по режиму, экраны победы/поражения для уровней.
