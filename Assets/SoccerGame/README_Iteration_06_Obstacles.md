# Iteration 6 — Препятствия

## Что изменилось с предыдущей итерации
- Добавлен `MovingObstacle.cs` — блок двигается между двумя точками
- Добавлен `RotatingObstacle.cs` — блок вращается вокруг оси
- Добавлен editor скрипт `SoccerGameSetup_Iteration6.cs`
- Все предыдущие скрипты без изменений

## Как настроить
1. Скопируй новые файлы в проект
2. В Unity: меню **SoccerGame → Setup Obstacles (Iteration 6)**

## Что создаётся на сцене
- **Obstacles** (родитель):
  - **StaticBlock_Left** — статичный блок слева (y=2)
  - **StaticBlock_Right** — статичный блок справа (y=2)
  - **StaticBlock_Center** — статичный блок по центру (y=5)
  - **MovingBlock_1** — двигается горизонтально в нижней зоне (y=-2)
  - **MovingBlock_2** — двигается горизонтально в верхней зоне (y=6.5)
  - **RotatingBlock** — вращающийся длинный блок в центре поля (y=0)

## Как тестировать
1. Нажми Play
2. Мяч должен отскакивать от всех препятствий
3. MovingBlock_1 и MovingBlock_2 двигаются влево-вправо
4. RotatingBlock крутится на месте
5. Траектория мяча стала более хаотичной — "пинбол хаос"
6. Матч всё ещё работает: счёт, голы, рестарт

## Ожидаемый результат
Поле заполнено препятствиями разных типов. Мяч отскакивает хаотично, создавая непредсказуемые ситуации. Забить гол стало интереснее — нужно учитывать отскоки.
