# Iteration 2 — Флипперы + управление

## Что изменилось с предыдущей итерации
- Добавлены `Flipper.cs`, `FlipperInput.cs`
- Добавлен editor скрипт `SoccerGameSetup_Iteration2.cs`
- Файлы из итерации 1 не изменены

## Как настроить
1. Скопируй новые файлы в проект
2. В Unity: меню **SoccerGame → Setup Flippers (Iteration 2)**
3. Флипперы и инпут создадутся автоматически, ссылки прокинутся

## Что создаётся на сцене
- **FlipperLeft** — левый флиппер внизу поля с HingeJoint2D
- **FlipperRight** — правый флиппер внизу поля с HingeJoint2D
- **FlipperInput** — объект с компонентом управления, ссылки на оба флиппера

## Как тестировать
1. Нажми Play
2. Клик/тап по левой половине экрана — левый флиппер бьёт вверх
3. Клик/тап по правой половине экрана — правый флиппер бьёт вверх
4. Мяч должен получать импульс при ударе флиппером
5. Флипперы возвращаются в исходное положение после отпускания

## Ожидаемый результат
Два флиппера V-образно внизу поля. Тап/клик по половине экрана активирует соответствующий флиппер. Мяч отбивается вверх при ударе.
