# Iteration 5 — Система счёта, Match Mode, UI

## Что изменилось с предыдущей итерации
- Обновлён `MatchManager.cs` — добавлена логика winScore=5, событие OnMatchEnd, флаг isMatchOver
- Добавлен `ScoreUI.cs` — отображение счёта через TMPro
- Добавлен `MatchEndUI.cs` — панель победы/поражения с кнопкой рестарта
- Добавлен editor скрипт `SoccerGameSetup_Iteration5.cs`
- Остальные скрипты без изменений

## Как настроить
1. Скопируй файлы в проект (заменить MatchManager.cs)
2. В Unity: меню **SoccerGame → Setup Score UI (Iteration 5)**
3. Canvas, счёт и панель конца матча создадутся автоматически

## Что создаётся на сцене
- **GameCanvas** — Canvas (ScreenSpaceOverlay, 1080x1920)
- **ScorePanel** — панель со счётом вверху экрана "0 - 0"
- **MatchEndPanel** — полноэкранная панель с результатом и кнопкой RESTART (скрыта)
- **EventSystem** (если не было)

## Как тестировать
1. Нажми Play
2. Счёт отображается вверху: "0 - 0"
3. При голе счёт обновляется
4. При 5 голах у любой стороны — появляется панель YOU WIN! или YOU LOSE! с финальным счётом
5. Кнопка RESTART — сбрасывает счёт, убирает панель, респаунит мяч

## Ожидаемый результат
Полноценный Match Mode: игра до 5 голов, счёт на экране, финальный экран с результатом и рестартом.
