# Iteration 14 — Туториал при первом запуске

## Что изменилось с предыдущей итерации
- Добавлен `TutorialUI.cs` — 5 слайдов с текстом, NEXT/SKIP, показывается один раз
- Обновлён `MainMenuUI.cs` — проверяет и показывает туториал при первом запуске
- Добавлен editor скрипт `SoccerGameSetup_Iteration14_Menu.cs`

## Как настроить
1. Скопируй файлы в проект (заменить MainMenuUI.cs)
2. В Unity: меню **SoccerGame → Setup Tutorial - Menu (Iteration 14)**
3. Откроется MainMenu, создастся TutorialPanel

## Что создаётся на MainMenu сцене
- **TutorialPanel** (внутри SafeAreaPanel если есть):
  - **Content** — тёмно-зелёный фон
  - **TitleText** — заголовок слайда
  - **BodyText** — текст слайда
  - **PageIndicator** — "1 / 5"
  - **NextButton** — зелёная кнопка NEXT (на последнем слайде — "GOT IT!")
  - **SkipButton** — серая кнопка SKIP (скрывается на последнем слайде)

## Слайды
1. **WELCOME!** — приветствие
2. **CONTROLS** — тап лево/право для флипперов
3. **SCORE GOALS** — забить мяч в верхние ворота
4. **DEFEND** — не пропускать мяч за флипперы
5. **GAME MODES** — описание трёх режимов

## Как тестировать
1. Удали ключ TutorialShown из PlayerPrefs (Edit → Clear All PlayerPrefs, или через код)
2. Запусти с MainMenu
3. Туториал появится автоматически с анимацией
4. Листай NEXT — слайды меняются с анимациями
5. SKIP — закрывает туториал
6. На последнем слайде SKIP скрывается, кнопка становится "GOT IT!"
7. После закрытия — при повторном запуске туториал не появляется

## Как сбросить туториал (для тестирования)
В Unity: Edit → Clear All PlayerPrefs
Или в коде: `PlayerPrefs.DeleteKey("TutorialShown");`

## Ожидаемый результат
При первом запуске игры появляется туториал с 5 слайдами. После просмотра или пропуска больше не показывается.
