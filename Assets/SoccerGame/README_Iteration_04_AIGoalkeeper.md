# Iteration 4 — AI Вратарь

## Что изменилось с предыдущей итерации
- Добавлен `AIGoalkeeper.cs` — простой AI вратарь
- Добавлен editor скрипт `SoccerGameSetup_Iteration4.cs`
- Файлы из итераций 1-3 без изменений

## Как настроить
1. Скопируй новые файлы в проект
2. В Unity: меню **SoccerGame → Setup AI Goalkeeper (Iteration 4)**

## Что создаётся на сцене
- **AIGoalkeeper** — красный прямоугольник перед верхними воротами, Kinematic Rigidbody2D, ссылка на мяч

## Как тестировать
1. Нажми Play
2. Вратарь двигается влево-вправо, следя за мячом с задержкой
3. При контакте с мячом — отбивает его вниз
4. Забить гол стало сложнее, но при хорошем тайминге возможно
5. Вратарь не выходит за пределы ворот (ограничен по X)

## Ожидаемый результат
Вратарь перед верхними воротами, двигается за мячом с небольшой задержкой реакции. Отбивает мяч вниз при контакте. Создаёт challenge для игрока.
